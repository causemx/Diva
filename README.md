DIVA is a lightning fast ground control station which extracts telemetry data from UAVs and execute tasks.

## Main Features

#### Data Extraction
Photon extracts the following data while crawling by default:

- URLs (in-scope & out-of-scope)
- URLs with parameters (`example.com/gallery.php?id=2`)
- Intel (emails, social media accounts, amazon buckets etc.)
- Files (pdf, png, xml etc.)
- JavaScript files & Endpoints present in them
- Strings based on custom regex pattern
